# CarQuery API

Wrapper for the [CarQuery](http://www.carqueryapi.com/) API in JavaScript.

## Installing
Using npm:
```
npm install car-query
```

## Example
```javascript
const carQuery = new CarQuery();
```

## API
```javascript
carQuery.getYears();
```

```javascript
carQuery.getMakes();
```

```javascript
carQuery.getModels();
```

```javascript
carQuery.getTrims();
```

```javascript
carQuery.getModelDetail();
```
